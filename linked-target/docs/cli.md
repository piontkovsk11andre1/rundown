# CLI Commands

Use this index to navigate command-specific CLI documentation.

## Command index

- `rundown add <seed-text> <markdown-file>` - append seed text to an existing Markdown file and run planning on it (`docs/cli-add.md`).
- `rundown loop <source>` - execute a task file repeatedly with iteration, cooldown, error-handling, and global runtime-limit controls (`docs/cli-loop.md`).
- `rundown plan <source>` - generate TODO workflows from a source prompt or markdown file (`docs/cli-plan.md`).
